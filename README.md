# Vijay Pathak

Personal blog at https://vijaypathak.com.np build
using [Minimal Mistakes](https://mmistakes.github.io/minimal-mistakes/).

## How to run?

- Build docker with ```docker build -t personal-website-jekyll .```
- Run using ```docker-compose up```
