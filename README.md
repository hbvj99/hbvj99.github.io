# Vijay Pathak - Personal Portfolio & Blog

Personal website and technical blog hosted at [vijaypathak.com.np](https://vijaypathak.com.np).

Built using **Astro v7** (Static Site Generator) for maximum speed, minimalist design, and clean developer workflows.

---

## 🛠️ Local Development

### Prerequisites
* Node.js `v22.12.0` or higher
* npm package manager

### 1. Install Dependencies
```bash
npm install
```

### 2. Run Local Development Server
Starts a hot-reloading dev server at [localhost:4321](http://localhost:4321):
```bash
npm run dev
```

### 3. Build Production Bundle
Compiles the static output inside the `dist/` directory:
```bash
npm run build
```

---

## 📁 Project Structure

* `/src/content/posts/` - Markdown articles frontmatter and post bodies.
* `/src/layouts/` - Astro page structures (BaseLayout, PostLayout).
* `/src/pages/` - Static site pages and dynamic sitemaps endpoint (`sitemap.xml.js`).
* `/public/` - Static production files (CNAME, robots.txt, ads.txt, assets/images, and favicons).
